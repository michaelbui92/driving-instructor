# Instagram Reels Production Guide

## 📂 Folder Structure

```
content/instagram/
├── raw/              # Original footage from phone camera
├── edited/           # Draft edits before final export
├── exports/          # Final render files ready for Instagram
├── scripts/          # Reel scripts and storyboards
├── graphics/         # Overlay graphics, overlays, animated elements
└── README.md         # This file
```

---

## 📋 Content Templates

### Script Format (Text File)

Located in: `scripts/`

Each script should include:
- Timing breakdown (seconds)
- Visual description
- Voiceover (if any)
- Caption/text on screen
- CTA placement

Example filename: `script-004-merge-zipper.txt`

Format:
```
REEL #4: Lane Merge - The Zipper
Length: 23 seconds
Format: Educational

[0-3s]   POV at merge point
Caption: Why everyone's doing lane merges WRONG

[3-10s]  Show cars cutting in aggressively
Voiceover: "Cutting in? That's rude AND dangerous"

[10-15s] Show zipper merge technique (alternating cars)
Voiceover: "Do THIS: Alternate lanes like a zipper"

[15-23s] Multiple POV angles showing zipper working
Voiceover: "It's not cutting in line. It's efficient driving."

[Endcard] Master the zipper merge → @drivewithbui

Hashtags:
#drivingtips #sydneydriving #drivinginstructor #learntodrive
```

## 🎨 Graphics Library

Located in: `graphics/`

1. **Arrow overlays** - PNG with transparency for showing direction

3. **Blind spot overlays** - outlines around mirror areas

## 🎬 Production Workflow

### 1. Pre-Production (Before Filming)

□ Read and practice script
□ Set up phone mount on dashboard
□ Check battery (50%+)
□ Clear phone storage
□ Choose lighting (natural daylight best)
□ Plan exact camera angles

### 2. Filming (Shoot Day)

□ Use filming locations:
   - Local parking lots (for practice shots)
   - Quiet streets (lower traffic)
   - Highway/motorway ramps (aneed permission/crew - use stock if necessary)
   
□ Capture requirements:
   - POV shots (windshield camera)
   - B-roll (mirror checks, hand movements)
   - Multiple takes if needed
   
□ Audio:
   - Voiceover recorded separately afterwards
   - Clear environment sound (wind noise minimal)
   - Consider lapel microphone for voiceover

### 3. Post-Production (Editing)

□ Import footage to editing software
□ Organize by script segment
□ Add audio:
   - Record voiceover
   - Add background music (royalty-free)
   - Mix levels
   
□ Edit:
   - Trim to timing in script
   - Add transitions (cuts only, no fancy effects)
   - Insert graphics overlays
   - Add captions/text
   - Color correction

□ Export:
   - Resolution: 1080x1920 (9:16 aspect ratio)
   - Frame rate: 60fps (smooth scrolling)
   - Codecs: H.264 (Instagram compatible)
   - Bitrate: 15-20 Mbps
   - Audio: AAC, 44.1kHz, 128kbps

### 4. Quality Assurance

□ Watch full video on phone (not just computer)
□ Check audio levels on phone speakers
□ Verify text is readable on small screen
□ Test Instagram upload
□ Check for any copyright music issues

## 📱 Instagram Upload Checklist

□ Export final version
□ Copy to phone (via AirDrop or Google Drive)
□ Open Instagram app
□ Choose "Create" -> "Reels"
□ Upload video file
□ Caption:
   - Hook text (1-2 sentences max)
   - CTA (link in bio / follow for more)
   - 2-3 relevant hashtags
   - @drivewithbui
   
□ Settings:
   - Cover photo (select best frame)
   - Location (optional)
   - Share to Facebook (recommended)
   - Share to Story (recommended)
   
□ Schedule or Post Immediately
   - Best times: Mon/Wed/Fri 12pm-3pm, 6pm-8pm
   - Avoid: Saturday mornings (test day?)

## 📏 Technical Specifications

### Video Export Settings:

```
Format: MP4
Codec: H.264
Resolution: 1080x1920 ((vertical video))
Frame Rate: 60fps (smooth playback)
Bitrate: 15000-20000 kbps
Audio Codec: AAC
Audio Bitrate: 128 kbps
Channels: Stereo
Sample Rate: 44.1kHz
```

### File Constraints:

- Max file size: 250MB per upload
- Max duration: 90 seconds (reels) / 60 seconds recommended
- File format: MP4 or MOV
- Aspect ratio: 9:16 (1080x1920 or 2160x3840)
- Audio: AAC-LC or AC3

## 🆘 Troubleshooting

### Video won't upload:

- Check file size (<250MB)
- Verify format (MP4/MOV)
- Test export with smaller resolution first
- Check Instagram storage quota

### Audio issues:

- Re-export with lower bitrate (100kbps)
- Remove background music temporarily
- Record voiceover separately

### Text not readable:

- Increase font size
- Use high contrast colors
- Test on phone screen before export
- Consider adding outlines/shadows to text

### Copyright strikes:

- Use royalty-free music
- Check YouTube Audio Library license
- Original voiceover is safest
